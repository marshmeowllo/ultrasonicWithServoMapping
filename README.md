# laughing-octo-tribble


## OLED Display with ESP8266
| OLED  | ESP8266 |
| ------------- | ------------- |
| Vin  | 3.3V  |
| GND  | GND  |
| SCL  | GPIO 5 or (D1)  |
| SDA  | GPIO 4 or (D2)  |

## HC-SR04 Ultrasonic Sensor with ESP8266
| HC-SR04  | ESP8266 |
| ------------- | ------------- |
| VCC  | VIN  |
| Trig  | GPIO 12 or (D6)  |
| Echo  | GPIO 14 or (D5)  |
| GND  | GND  |
